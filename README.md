# Installation

- Clone the repository:

```sh
git clone https://github.com/onaki88/game-of-shells.git
```

- Install the node dependencies:

```sh
cd game-of-shells
npm install
```

- Start your local node server:

```sh
  npm start
```

- Generate the build

```sh
  npm build
```
